# Student Learning Analytics Dashboard

## Project Overview

This project analyzes student academic performance and attendance using Excel and Power BI.

## Objectives

* Track student performance across subjects
* Monitor attendance trends
* Identify at-risk students
* Generate actionable insights for educators

## Tools Used

* Microsoft Excel
* Power BI Desktop

## Key Metrics

* Total Students
* Average Marks
* Average Attendance
* Subject-wise Performance
* Performance Distribution
* At-Risk Students

## Insights

* Average student marks were 55.
* Average attendance was 73%.
* Science and Computer subjects showed relatively better performance.
* Students with low attendance and low marks were identified as at-risk.

## Dashboard Features

* Interactive Class Filter
* Subject Filter
* KPI Cards
* Performance Charts
* Attendance Analysis
* At-Risk Student Tracking

* Data Cleaning
* Data Transformation
* Data Visualization
